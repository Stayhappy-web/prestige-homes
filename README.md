# Prestige Homes

A modern and responsive real estate website built with React and Vite, designed to showcase luxury properties, premium homes, and investment opportunities through an elegant and user-friendly interface.

## Features

* Modern and responsive design
* Property search section
* Featured property listings
* Detailed property pages
* Image galleries for each property
* Property specifications and descriptions
* Professional statistics section
* About and Contact sections
* Mobile-friendly navigation
* React Router integration
* Clean and scalable code structure

## Technologies Used

* React.js
* Vite
* React Router DOM
* CSS3
* JavaScript (ES6+)

## Getting Started

Clone the repository:

git clone https://github.com/your-username/prestige-homes.git

Navigate into the project folder:

cd prestige-homes

Install dependencies:

npm install

Start the development server:

npm run dev

Build for production:

npm run build
